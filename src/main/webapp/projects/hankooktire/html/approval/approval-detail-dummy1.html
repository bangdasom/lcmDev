제목
한국동그라미파터너스 (HDP) 품의/보고 양식 개발 요청
귀 팀의 업무협조에 감사드립니다.
한국동그라미파트너스 (HDP) 의 품의서/보고서 개발을 아래와 같이 요청드립니다.

- 아 래 -

1. HDP 품의서/보고서 개발 요청 배경
- AS-IS : HDP 의 업무가 상당수 HKT 예산을 사용하여 집행하고 있으며, 품의서/보고서는 HDP 의 전결 & HKT 합의로 진행
→ HKT 의 예산을 사용하는 업무가 대다수이기에, HKT 에서 책임소재를 지기 위해선 HKT 전결 품의서/보고서 필요
→ HDP 및 HKT 의 결재를 동시에 得하면서, 업무 Loss 를 최소화 하기 위하여 HDP 품의서/보고서 개발 요청

2. HDP 품의서/보고서 개발 요청 내용
- 예산 신청서와 같은 양식의 품의서/보고서 개발 요청
1) HDP 에서 품의서/보고서 작성, 주관 부서는 예산 보유 팀으로 지정
2) HDP 에서 품의서/보고서결재 완료
3) HDP 품의서/보고서 결재 완료 이후, 품의서/보고서가 예산 보유 팀으로 발송
4) 주관부서에서 품의서/보고서 접수, HKT의 전결 합의 권한에 따라 재 상신
→ 이해를 돕기 위하여 예산신청서 하나를 Sample 로 함께 첨부드립니다.
5) 품의 작성 란은 현재 개발중인 일반 HKT용 품의서/보고서와 동일한 양식으로 적용바랍니다.

- 향후 HDP의 업무 확장을 감안하여 운영될 예정인 DWP의 통합적인 시스템에 본 HDP 품의 양식을 동일하게 반영 요청,
또는 향후 반영 가능한 Room 마련 요청드립니다.

3. 개발 완료 및 시행 요청 날짜 : 2017년 1월 DWP Open 시.

4. 첨부 문서 : 예산 신청서 (sample)
