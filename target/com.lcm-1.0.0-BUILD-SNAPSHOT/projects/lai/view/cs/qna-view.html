판매자가 반품된 상품을 받고 아무런 이상이 없다 확인한 후
반품완료로 처리한 후부터 환불이 진행됩니다.

단, 구매자의 사유로 반품 시 판매자에게 반품배송비가 필요합니다.

이때 반품 배송료를 지불하지 않았다면
반품이 보류되어 반품 승인이 지연될 수 있습니다.
